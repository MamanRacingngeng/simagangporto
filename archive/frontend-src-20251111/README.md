Magang Digital BBKB — React Dashboard (prototype)

This folder contains a small React + Vite + Tailwind prototype of the Magang Digital BBKB dashboard.

Features:
- Sidebar navigation
- Header with notification and avatar
- Status card with indicators and decorative art area
- Quick action cards (Upload, Details)
- Framer-motion entry animation

How to run:

1. Open a terminal in this folder: `d:\Projectbatika\frontend`
2. Install dependencies:

   npm install

3. Run dev server:

   npm run dev

The app will be available on http://localhost:5173 by default.

Notes:
- I used Tailwind for styling and framer-motion for small animations.
- The design aims to follow the prototype you provided: modern, soft shadows, rounded cards and blue primary color.
- You asked to use ShadCN UI — I implemented components using Tailwind to mimic ShadCN styling. If you want to integrate the official ShadCN component library, I can adapt the project accordingly.
